# Layoffs Data Analysis - SQL Data Cleaning & EDA

Complete data analysis project analyzing layoffs data using SQL.

## Dataset
- **layoffs.csv**: Raw dataset with company layoffs, funding, employee counts (170K+ rows)

## Analysis Steps

### 1. Data Cleaning (`1.Data-Cleaning.sql`)
- Removed duplicates and nulls
- Standardized company names and dates
- Handled inconsistent employee count formats

### 2. Exploratory Data Analysis (`2.EDA-Exploratory-Data-Analysis.sql`)
- Calculated layoff trends by year, industry, company size
- Identified top companies by layoffs and funding raised
- Analyzed layoff rates vs employee count and funding rounds
